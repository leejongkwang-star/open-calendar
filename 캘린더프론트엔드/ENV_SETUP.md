# 프론트엔드 환경 변수 설정

## .env 파일 생성

프론트엔드 루트 디렉토리에 `.env` 파일을 생성하고 다음 내용을 추가하세요:

```env
# 백엔드 API 기본 URL
VITE_API_BASE_URL=http://localhost:3001/api
```

## 설정 방법

1. `캘린더프론트엔드` 폴더에 `.env` 파일 생성
2. 위 내용을 복사하여 붙여넣기
3. 프론트엔드 서버 재시작

## 참고

- `VITE_API_BASE_URL`이 설정되어 있으면 자동으로 실제 백엔드 API를 사용합니다
- 설정하지 않으면 Mock 데이터를 사용합니다
- 환경 변수는 `VITE_` 접두사가 필요합니다 (Vite 요구사항)
